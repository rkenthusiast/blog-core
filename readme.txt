Enhance your blog theme with powerful Gutenberg blocks for creating engaging content.

== Description ==

Blog Core is a WordPress plugin that provides a collection of Gutenberg blocks specifically designed to enhance blog themes. With Blog Core, you can easily create visually stunning and interactive content for your blog posts.

Features:
- Includes a variety of Gutenberg blocks tailored for blog content.
- Customize block styles and settings to match your theme's design.
- Intuitive block editor interface for easy content creation.
- Regular updates and new blocks added over time.

== Installation ==

1. **Install Dependencies**: Before uploading the plugin, make sure to install any dependencies required by running the following command in the plugin directory:
   ```bash
   npm install

2. Install Composer Dependencies: Before uploading the plugin, make sure to install any dependencies required by running the following command in the plugin directory: 
    ```bash 
    Composer install

2. Upload Plugin: Upload the blog-core folder to the /wp-content/plugins/ directory of your WordPress installation.

3. Activate Plugin: Activate the plugin through the 'Plugins' menu in WordPress.

4. Add Blocks: After activation, use the Gutenberg editor to add Blog Core blocks to your blog posts and pages.
Available Blocks After:
-- Homepage - Banner
-- Homepage - Posts
-- Homepage - Footer
-- Homepage - Advertisement


== Frequently Asked Questions ==

= How do I use Blog Core blocks in my blog posts? =

Once the plugin is activated, navigate to the Gutenberg editor when creating or editing a blog post. Click the "+" icon to add a new block, then search for "Blog Core" to find and add the available blocks to your content.

= Can I customize the appearance of Blog Core blocks? =

Yes, you can customize the appearance and settings of Blog Core blocks to match your theme's design. Each block comes with various customization options accessible from the block toolbar or block settings panel.
